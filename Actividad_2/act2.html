Instrucciones


  • Desarrollar un programa para ver los servicios de Windows y Linux.


Requerimientos


  • La última versión de Python instalado en ambos Sistemas Operativos.



Comencemos




  1. Lo primero será instalar Python en Windows 10. A la fecha, la ultima versión corresponde a la 3.8.1, y puedes acceder a la liga de descarga aquí.


  2. Nos dirigiremos a la parte inferior, y seleccionaremos la opcion Windows x86-64 executable installer si tu computadora es de 64 bits. Si tu computadora es de 32 bits, deberás elegir la opción Windows x86 executable installer.




    3. Si no estás seguro acerca de los bits de tu computadora, puedes consultarlo dando clic derecho en el botón de inicio de Windows, seleccionando la opción Sistema, y en el campo denominado Tipo de sistema podrás verificar este dato.





    Si el instalador solo nos muestra las opciones de reparar o desinstalar, querrá decir que ya tenemos instalado Python.




  3. Nos aseguramos de tener seleccionado el recuadro de Add Python to PATH,ya que nos facilitaría bastantes procesos en un futuro, para todo tipo de cosas. Y posteriormente seleccionamos Instalar ahora. Una vez terminada la instalación, cerramos el asistente de instalación sin más.

    4. Ahora procederemos a instalar Python en Linux - Ubuntu.
    En sistemas operativos como Ubuntu, la instalación de programas suele ser completamente distinta a las de Windows. En este caso tendremos que recurrir a la línea de comandos (también conocida como consola o terminal) para poder proceder.



  4. Lo que haremos será abrir la terminal. Para esto podemos presionar el boton de Windows de nuestro teclado y escribir Terminal, o dar clic en el menú de opciones ubicado en la esquina inferior izquierda y escribir la misma opción mencionada.
